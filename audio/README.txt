Place your audio files here.

Expected filenames (you can swap formats, e.g., .ogg/.mp3):
- bgm_default.mp3
- bgm_plaza.mp3
- bgm_ranch.mp3
- bgm_maze.mp3

Optional SFX files (if you want to replace generated synth SFX):
- sfx_kill.mp3
- sfx_pickup.mp3
- sfx_levelup.mp3
- sfx_hit.mp3
- sfx_start.mp3
- sfx_death.mp3
- sfx_menu.mp3
- sfx_alert.mp3

If these files exist, the front-end will use them via the ES module `public/js/audio.js` for BGM.
